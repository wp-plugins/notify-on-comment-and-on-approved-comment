Send a notification on comment to an specified email to approve the post
and another notification to another specified email when post is approved

== Description ==

This is a very simple plugin that sends notifications to an specified email when someone writes a comment to the post to approve the post.
When post is approved, this plugin sends notificacions to another specified email.

== Installation ==

1. Upload 'notify-on-comment' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all
